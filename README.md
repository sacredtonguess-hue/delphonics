# The Delphonics Archive

The Delphonics Archive is a structured linguistic project focused on reconstructing glossolalia as a coherent language system. This repository documents the use of Sanskrit, Prakrit, and Marathi frameworks to romanize, translate, and preserve the narrative structures embedded within speaking in tongues.

---

## Purpose

This project operates on the premise that glossolalia is not random or unknowable speech, but a recoverable linguistic system. By applying Indo-Aryan grammatical frameworks, the archive seeks to establish a repeatable method for translation and interpretation.

---

## Structure

- /audio → Source recordings (segmented + full)
- /translations → Completed translated passages  
- /lexicon → Structured word database  
- /methodology → Translation rules and grammar system  
- /notes → Working interpretations and drafts  

---

## Translation Format

Each entry follows a consistent three-line structure:

1. Glossolalia (italicized)  
2. Devanagari script  
3. English translation  

Example:

Bai yān naukau Yarada ar sur makhin nekki hay
बाई यान नौकौ यारदा अर् सुर् मखिन् नेक्की हय
the Mother’s Phalanx of paired Emissary–Yarada, who killed the Priest of the Upright Horse

---

## Methodology

- Romanized input is treated as spoken Marathi/Prakrit  
- Final vowel drops are accepted (spoken-form logic)  
- Words are mapped to Sanskrit/Prakrit roots  
- Meaning is derived through grammar and structure, not intuition  

---

## Lexicon Standard

All lexical entries follow this format:

Devanagari | Romanization | Language | Root | Grammatical Form | Definition
